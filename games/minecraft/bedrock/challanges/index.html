Minecraft but You Spawn Where You Die

This challenge requires a single Command Block to set up. You can put the command block anywhere. Just also make sure you put a Ticking Area where the Command Block is. To do that you use the /tickingarea command

Command Block Settings

Block Type: REPEAT
Output Signal: ALWAYS
Activation: ALWAYS ACTIVE
Activate on first tick: TRUE
Tick Speed: 0
Command: execute as @a at @s spawnpoint @s ~ ~ ~

Another way you can do this *optional*

Change the command in the command block with this (execute as @a at @s spawnpoint @s ~ ~10 ~)
